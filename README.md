
# GreenHomes Store

A modern e-commerce platform built with **WordPress**, **WooCommerce**, and hosted on **Pantheon**.

GreenHomes Store is an online marketplace dedicated to bringing nature indoors by providing a curated selection of indoor and garden plants. The website is designed to provide customers with a smooth, secure, and enjoyable shopping experience through an attractive interface, organized product categories, and responsive design.

# Project Overview

GreenHomes Store was developed as an e-commerce website to demonstrate the practical implementation of online business concepts using WordPress and WooCommerce.

The website allows plant enthusiasts to browse products, search for various plant species, view detailed descriptions, and manage their purchases through a simple and user-friendly interface.

The primary objective of this project is to create an online botanical store that combines attractive design with efficient e-commerce functionality while applying WebOps deployment practices using Pantheon.

---

# Objectives

The project aims to:

- Build a professional online plant and garden store.
- Provide customers with a convenient, nature-focused shopping experience.
- Showcase indoor and garden plants in an organized manner.
- Demonstrate the practical use of WordPress and WooCommerce.

# Features Implemented

## Responsive Design

The website is fully responsive, allowing customers to shop comfortably using desktops, tablets, and smartphones.

## Product Catalog

Products are organized into categories such as:

- Indoor Plants
- Garden Plants
- Planting Accessories
- Featured Collections

Each product includes:

- Product image
- Product name
- Price
- Description
- Category

## Shopping Cart

Customers can:

- Add products to the cart
- Update quantities
- Remove items
- View total pricing before checkout

## Product Search

The website provides a search feature for customers to quickly find specific plant varieties or tools.

## WooCommerce Integration

WooCommerce manages:

- Products
- Inventory
- Orders
- Customer information
- Pricing

## Fast Performance

The Astra theme and Pantheon hosting improve loading speed, making it ideal for displaying high-quality plant images.

## Secure Deployment Workflow

Pantheon's **Dev → Test → Live** workflow was used to safely develop and deploy website updates.

# Challenges Faced

## 1. Migration Hurdles

Dealing with browser-based upload limits during the site migration process required finding efficient ways to move the 320 MB website.

## 2. Git Configuration

Properly configuring the `.gitignore` file ensured that only custom code was tracked while excluding WordPress core files and sensitive configuration data.

## 3. Environment Syncing

Learning Pantheon's **Dev → Test → Live** workflow and understanding how code moves separately from database content was essential for smooth deployment.
