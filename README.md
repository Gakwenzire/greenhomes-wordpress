## Project Overview
**GreenHomes Store** is an online marketplace designed to provide a seamless shopping experience for plant enthusiasts. The platform allows users to browse through various categories of indoor and outdoor plants, view detailed descriptions, and manage their purchases.

* **Platform Used:** WordPress
* **Hosting Provider:** Pantheon (WebOps Platform)
* **Theme:** Astra, Woocommerce

---

## Features Implemented
* **User-Friendly Interface:** Responsive design powered by the Astra theme.
* **Product Catalog:** Categorized listing of indoor plants, garden plants, and accessories.
* **Easy Navigation:** Intuitive search and menu structures for quick product discovery.
* **Fast Performance:** Optimized for web speed utilizing Pantheon's Global CDN.
* **Secure Deployment:** Professional WebOps workflow (Dev → Test → Live).
